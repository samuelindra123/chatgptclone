# Xynoos AI UI

A production-quality Next.js UI for Xynoos AI, tuned for dark mode by default and built with local mock data.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Zustand
- lucide-react
- Framer Motion

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Notes

- No backend is required.
- The UI uses local mock conversations, models, and attachment previews.
- The layout is desktop-first and collapses the sidebar on smaller screens.
